    Schéma de la base de données d'Open Planner

    Afin de faciliter le travail des développeurs intéressés par le projet, cette rubrique va s'attacher à décrire la structure de la base de données d'Open Planner.

    Ce projet a été conçu pour être utilisé avec un serveur de base de données PostgreSQL dont la version doit être >= 9. Le nom de la base de données est planner. Ce nom n'est pas personnalisable.
